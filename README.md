# Modelo-clasificacion
Se desarrolla un modelo que permitirá la clasificacion de personas a partir de imagenes de rostros si una persona es joven o vieja

 # Proposito
 Desarrollar un modelo de clasificación basado en aprendizaje profundo capaz de distinguir entre rostros jóvenes y adultos mayores a partir de imágenes, con el fin de explorar su utilidad en aplicaciones biomédicas relacionadas con la estimación automatizada de la edad. Este sistema busca aportar herramientas de apoyo que faciliten la evaluación de indicadores asociados al envejecimiento, la detección temprana de cambios faciales vinculadas a condiciones de salud y la investigación sobre patrones biométricos relevantes en poblaciones clínicas.
